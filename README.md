# CAPPA SQLPlag
SQL-детектор, предназначенный для использования в комплексе автоматической проверки программ [CAPPA](http://cappa.csu.ru/).

### Установка
 
```shell
pip install cappa_sqlplag
```

[Пример использования](docs/usage.md)
